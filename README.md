点击“导入WAV文件”按钮，选中导入文件后，调用wavHandle.startReadFromWav()；并传入选择的文件的路径，同时打开分析报告页面。
wavHandle.startReadFromWav()；函数会创建一个新的线程，在新线程中首先调用importFromWav函数，该函数调用对应的WAV文件解码python脚本，完毕后发出importDataReady(voltageData)信号，该信号会调用SignalPreprocessiong实例中的filterDataImport函数，filterDataImport函数会对数据进行带通滤波与小波降噪，然后返回处理后的数据，更新wavHandle内的m_timeDomainDataImport并发出importDataReady信号。DataVisualizationGraphCartesianSpectrum绑定了importDataReady信号，当接受到这个信号后会调用signalDFTCalculation.dftDataImport();函数，这个函数会内会调用WAVHandle单例中的timeDomainDataImport();函数以获取经过带通滤波与小波降噪的数据，然后对其做离散傅里叶变换获得频域的点阵，再调用toSpectrumPointList函数将点阵存储到QVariantList返回到qml，最终绘制出频域图像。
OSDataVisualizationGraphCartesianWaveform绑定了wavHandle的importDataReady信号，当接受到这个信号后会调用adaptiveDownsampling.downsamplingDataImport();后续处理同上。

点击 “开始采集”按钮，调用daqManager.startCollection，此后数据采集卡会不断采集数据并将当前数据刷新（不会保存数据，每次都会将最新数据覆盖旧数据）

点击“开始绘制”按钮，
调用signalPreprocessing.startPreprocessing();函数，该函数内启动定时器依次调用processing()；函数，该函数通过调用DaqDeviceManager单例的getDataCache函数获取对应通道的采集数据，对其做带通滤波与小波降噪后调用WAVHandle单例的splicTimeDomainData函数，将处理后的数据拼接。Processing函数处理完毕后发出resultReady信号，刷新SignalPreprocessing的m_preprocessedDataCache。
调用adaptiveDownsampling.startDownsamplingProcessing();该函数发出processingStart信号，处理完毕后发出resultReady信号，更新AdaptiveDownsampling内的m_downsampledData并发出downsampledDataReady信号通知qml侧数据更新，自动同步最新数据
